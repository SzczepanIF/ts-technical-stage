Tigerspike Tech Task

Author: Marcin Szczepanczyk

Technologies used:

Typescript, Gulp, SASS, PUG, tslint karma, jasmine, npm, es5/6 shims

npm tasks (most running via gulp):

npm run start
putting server live and opening browser (via browsersync)
npm run test
starts unit tests via karma runner
npm run lint
runs typescript linter over ts files
npm run build
creating new build copied to the `dist` folder
npm run no-pug
updating pug files from no-pug directory (so. basically the partials, which are not separate compiled template files)
